ROUTING CONFIGURATION (HYBRID)
---------------------------------------------------------------------------
! 1. OSPF ke Arah Core
router ospf 1
 network 192.168.5.12 0.0.0.3 area 0
 redistribute static subnets
 exit

! 2. Static Route ke Lantai (Summary Routes)
! Mengarahkan paket ke masing-masing router lantai
ip route 192.168.8.0 255.255.255.128 192.168.5.22   ! Lt 1
ip route 192.168.8.128 255.255.255.128 192.168.5.26 ! Lt 2
ip route 192.168.9.0 255.255.255.128 192.168.5.30   ! Lt 3 (Expanded /25->/24 logic applied)
ip route 192.168.9.128 255.255.255.128 192.168.5.34 ! Lt 4
ip route 192.168.10.0 255.255.255.128 192.168.5.38  ! Lt 5

! *Note: Untuk routing static lantai, pastikan sesuai dengan tabel routing
! yang ada di router lantai masing-masing.
